Project Name:

Rename Files

Description:

This script copies all the files in one directory, creates copies in a new directory and the files are renamed to a "Prefix-#" name structure

Usage:

Note: Written and tested in Pyton 3.7

Note: This script was written for Windows.

Installation:

This script is written in the main() feature.
